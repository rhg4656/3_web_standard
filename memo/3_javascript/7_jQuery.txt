▷ jQuery
	자바스크립트를 더 쉽고 간편하게 사용하기 위해 만들어진 라이브러리.

▶ $
	$ 은 jQuery 객체를 가리킬 때 사용한다.
	$(선택자) -> DOM 요소를 선택하고 메소드를 호출한다.

▶ DOM 조작
	.text() : 요소의 텍스트
	.val() : 폼 요소의 값
	.attr() : 속성 값

	.on('이벤트 이름', 콜백함수) : 이벤트 추가

	.next() : 다음 요소
	.prev() : 이전 요소

	.append() : 괄호 안 문자열 html 에 추가
	.remove() : 요소 삭제

	.addClass : 클래스 추가
	.removeClass : 클래스 삭제
	.toggleClass : 클래스 추가 삭제 반복
