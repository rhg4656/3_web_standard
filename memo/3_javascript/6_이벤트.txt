▷ 이벤트
	웹 브라우저와 사용자 사이에 상호작용이 발생하는 특정 시점.
	이벤트가 발생하면 이벤트 종류에 따라 어떤 작업을 하거나 함수를 호출하는 등의 조작을 지정할 수 있다.

▶ 이벤트 종류 (200 여가지가 넘음.)
	1) 마우스 이벤트
		- onclick : 마우스 클릭시 발생
		- ondblclick : 마우스 더블 클릭시 발생
		- onmouseover : 마우스 포인터를 올리면 발생
		- onmouseout : 마우스 포인터가 빠져나가면 발생
		- onmouseleave : 마우스 포인터가 빠져나가면 발생 (자식으로 갈 시에는 이벤트 발생X)
		- onmousemove : 마우스 포인터가 움직이면 발생
		- onwheel : 마우스 휠을 움직이면 발생

	2) 키보드 이벤트
		- onkeypress : 키보드 버튼을 누르고 있는 동안 발생
		- onkeydown : 키보드 버튼을 누른 순간 발생
		- onkeyup : 키보드 버튼을 눌렀다가 뗀 순간 발생

	3) 포커스 이벤트
		- onfocus : 요소에 포커스가 되면 발생
		- onblur : 요소가 포커스를 잃으면 발생

	4) 폼 이벤트
		- onsubmit : 폼이 전송될 때 발생

▶ 이벤트 등록
	1) 인라인 방식
		html 요소에 속성처럼 추가
		속성 이름은 이벤트 이름, 값은 함수

	2) 요소 노드에 직접 등록
		요소 노드를 선택 후 노드의 속성으로 이벤트를 등록

	3) 이벤트 등록 메소드 사용 ★
		DOM 에서 제공하는 addEventListerner() 메소드를 사용해 등록

▶ 이벤트 취소
	태그나 요소에 기본적으로 지정되어있는 이벤트를 취소할 수 있다.
	form 태그의 submit 버튼, a 태그 등
	
	e.preventDefault() -> 이벤트 취소
